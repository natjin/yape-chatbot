# Yape Chatbot Prototype — Yapito

A mobile chatbot prototype for **Yape**, Peru's leading mobile payment app. Features "Yapito," a friendly FAQ assistant with Yape branding.

## Features

- **Mobile-first UI** — phone frame mockup on desktop, full-screen on mobile
- **10 FAQ topics** with humorous, direct responses
- **Financial advice guardrail** — politely declines investment questions
- **Typing indicator** with bouncing dots animation
- **Quick-reply chips** for easy navigation
- **Pattern matching** on keywords with rotating fallback responses
- **Peruvian flavor** — Spanish phrases like "¡Yapea!" and "¡Eso es todo!"

## Topics Covered

1. How to send money
2. Fees and pricing
3. Transfer limits
4. Account creation
5. Security
6. International use / remittances
7. Yape Tienda (marketplace)
8. Investment advice (guardrail)
9. Troubleshooting
10. Company history (BCP)

## Tech Stack

- Vanilla HTML, CSS, JavaScript — no dependencies
- Google Fonts (Nunito)
- Fully static — no backend required

## Running Locally

Just open `index.html` in your browser, or serve with any static file server:

```bash
npx serve .
```
